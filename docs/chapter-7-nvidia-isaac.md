# Chapter 7: NVIDIA Isaac AI & Robot Brain

## What is NVIDIA Isaac?
- NVIDIA Isaac is a platform to **train and control robots using AI**  
- It allows robots to:
  - See the environment  
  - Understand objects  
  - Plan movements  

---

## Isaac Sim
- A **simulator for training robots**  
- Can create **photorealistic environments**  
- Helps generate **synthetic data** for AI training  

---

## Isaac ROS
- Hardware-accelerated AI for **real robots**  
- Includes **VSLAM** (Visual SLAM) for navigation  
- Helps robots move and avoid obstacles  

---

## Nav2 (Navigation)
- Plan paths for robots  
- Helps humanoid robots **walk or reach a target**  

---

## Summary
- NVIDIA Isaac = robot brain software  
- Isaac Sim → train in virtual world  
- Isaac ROS → deploy AI on real robots  
- Nav2 → robot path planning
