# Physical AI & Humanoid Robotics

This project is a beginner-friendly textbook for learning Physical AI and Humanoid Robotics.

The book explains how artificial intelligence works in the real physical world using robots.
It covers topics like:
- Physical AI
- Robotics basics
- ROS 2
- Simulation
- Humanoid robots
- Vision, language, and action

This project is being built for a hackathon organized by Panaversity.
